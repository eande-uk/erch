# Hotkeys

## System

| Key | Action |
|-----|--------|
| Super + Escape | Lock screen |
| Super + Shift + E | Logout menu |
| Super + Shift + S | Shutdown menu |
| Ctrl + Alt + Del | Reboot |
| Super + Shift + R | Reload Hyprland config |

## Launcher & Menus

| Key | Action |
|-----|--------|
| Super + Space | Application launcher (Walker) |
| Super + Alt + Space | erch menu |
| Super + R | Resize mode |
| Super + Shift + Return | Terminal |

## Windows

| Key | Action |
|-----|--------|
| Super + Q | Close focused window |
| Super + F | Toggle fullscreen |
| Super + Shift + Space | Toggle floating |
| Super + J | Toggle split direction |
| Super + Backspace | Toggle layout |
| Super + Tab | Cycle through open windows |
| Super + Grave (`) | Cycle through windows on all workspaces |

## Moving & Resizing

| Key | Action |
|-----|--------|
| Super + R (then arrows) | Resize mode |
| Super + mouse drag | Move floating window |
| Super + right-click drag | Resize floating window |

## Workspaces

| Key | Action |
|-----|--------|
| Super + 1–9, 0 | Switch to workspace 1–10 |
| Super + Shift + 1–9, 0 | Move window to workspace 1–10 |
| Super + Ctrl + 1–9, 0 | Move window to workspace and follow |
| Super + mouse scroll | Cycle through workspaces |
| Super + Alt + 1–9 | Switch to workspace (with gaps for visibility) |

## Screenshots & Recording

| Key | Action |
|-----|--------|
| Print Screen | Fullscreen screenshot → clipboard |
| Shift + Print Screen | Region screenshot → clipboard |
| Alt + Print Screen | Focused window screenshot → clipboard |
| Ctrl + Print Screen | Fullscreen screenshot → save |
| Ctrl + Shift + Print Screen | Region screenshot → save |
| Super + Shift + R | Screen recording region |
| Super + Alt + R | Screen recording fullscreen |

## Clipboard

| Key | Action |
|-----|--------|
| Super + C | Copy file path or screenshot to clipboard |
| Super + V | Paste from clipboard history |
| Super + Shift + C | Open clipboard manager |

## Quick Actions

| Key | Action |
|-----|--------|
| Super + B | Hide/show Waybar |
| Super + Shift + T | Toggle scratchpad terminal |
| Super + M | Toggle mute |
| Super + . | Emoji picker |
| F16 (if available) | Toggle keyboard backlight brightness |
