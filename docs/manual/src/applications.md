# Applications

erch ships with a curated set of applications across categories. All are free and open source unless noted.

## Terminal & Shell

| App | Purpose | Launch |
|-----|---------|--------|
| Alacritty | Primary terminal | `Super + Return` |
| Kitty / Foot / Ghostty | Alternative terminals | `erch launch terminal` |
| Starship | Prompt theme | Auto-configured |
| zoxide | Smarter cd | `z <dir>` |
| fzf | Fuzzy finder | `Ctrl+T`, `Ctrl+R` |
| bat | Better cat | `bat <file>` |
| fd | Better find | `fd <pattern>` |
| ripgrep | Better grep | `rg <pattern>` |
| tmux | Terminal multiplexer | `tmux` (auto-start in Alacritty) |
