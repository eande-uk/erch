# Navigation

erch is keyboard-driven. The Super key (Windows key on most hardware, Command on Mac) is your primary modifier.

## Application Launcher

Press **Super + Space** to open Walker, the application launcher. Start typing to search for apps, files, or commands.

## erch Menu

Press **Super + Alt + Space** to open the erch menu. This gives you quick access to:

- Theme switching
- Screenshot tools
- Power options (lock, logout, reboot, shutdown)
- System settings
- Clipboard and file sharing

## Window Management

| Action | Shortcut |
|--------|----------|
| Close window | Super + Q |
| Fullscreen | Super + F |
| Toggle floating | Super + Shift + Space |
| Move to workspace | Super + 1–9 |
| Move window to workspace | Super + Shift + 1–9 |
| Resize mode | Super + R (then arrows) |
| Toggle split | Super + Backspace |
| Cycle layouts | Super + Tab |

## Virtual Desktops (Workspaces)

Workspaces are arranged horizontally by default. You can have up to 10 (Super + 1–9 and Super + 0 for workspace 10).

- Switch workspace: **Super + Number**
- Move window to workspace: **Super + Shift + Number**
- Move to empty workspace: **Super + Shift + 0**

## Resizing Windows

Press **Super + R** to enter resize mode. Use arrow keys or HJKL to resize the focused window. Press **Super + R** or **Escape** to exit resize mode.
