# Configuration

## File Locations

erch separates configs into two locations:

- **`~/.config/`** — your configs. Edit these freely.
- **`~/.local/share/erch/`** — erch's defaults. **Do not edit** — they are overwritten on update.

To restore a default config:

```bash
erch refresh waybar       # Reset Waybar config to default
erch refresh hypr/hyprlock.conf  # Reset specific file
```

This backs up your current config to `.bak.<timestamp>` before restoring.

## Themed Configs

Some configs contain theme variables and are re-generated every time you switch themes:

| Config | Template | Generated by |
|--------|----------|-------------|
| `~/.config/alacritty/alacritty.toml` | `default/themed/alacritty.toml.tpl` | `erch theme set` |
| `~/.config/waybar/style.css` | `default/themed/waybar.css.tpl` | `erch theme set` |
| `~/.config/mako/config` | `default/themed/mako.ini.tpl` | `erch theme set` |
| `~/.config/erch/current/theme/erch.css` | `default/themed/erch.css.tpl` | `erch theme set` |
| `~/wiki/theme/css/variables.css` | `default/themed/mdbook-variables.css.tpl` | Symlinked on `erch wiki notes init` |

To customize a themed config without losing it on theme change, use the user override:

```bash
mkdir -p ~/.config/erch/themed
cp ~/.local/share/erch/default/themed/waybar.css.tpl ~/.config/erch/themed/
# Edit ~/.config/erch/themed/waybar.css.tpl — your changes survive theme switches
```

## Hooks

Hooks are shell scripts that run at specific points in erch's lifecycle:

| Hook | When it runs |
|------|-------------|
| `~/.config/erch/hooks/theme-pre.sh` | Before theme switch |
| `~/.config/erch/hooks/theme-post.sh` | After theme switch |
| `~/.config/erch/hooks/update-pre.sh` | Before update |
| `~/.config/erch/hooks/update-post.sh` | After update |

Create any of these files and erch will source them automatically.

## Environment Variables

erch sets sensible defaults for common environment variables in `~/.config/erch/env`. You can override any of them:

```bash
export EDITOR=nvim          # Already set by default
export BROWSER=firefox      # Override the default Chromium
export TERMINAL=kitty       # Override the default Alacritty
```

## Monitors & Display

```bash
# List connected monitors
hyprctl monitors

# Open the monitor configuration
nvim ~/.config/hypr/monitors.conf
```

Format:

```conf
monitor=,preferred,auto,1        # Auto-detect main monitor
monitor=DP-1,1920x1080@144,0x0,1 # Manual configuration
monitor=eDP-1,disable            # Disable laptop screen
```

## Keybindings

All keybindings are in `~/.config/hypr/bindings.conf`. Edit this file to add or change shortcuts, then reload with `Super + Shift + R`.
