# Security Policy

## Reporting a vulnerability

Do not open a public issue. Report privately by emailing
security@eande.uk or contacting the maintainers directly.

You should receive a response within 48 hours. If the issue is confirmed,
a fix will be released as soon as possible.

## Scope

This covers the erch operating system configuration and all scripts in
this repository. For upstream dependencies (Linux kernel, packages),
report to the respective project.
