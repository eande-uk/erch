# config/waybar/ — Waybar Status Bar

Waybar status bar configuration: top bar layout, modules (clock, workspaces, system tray, battery, network, audio), and theme-aware CSS.

## Contents

| File | Purpose |
|------|---------|
| `config.jsonc` | Module definitions and bar layout |
| `style.css` | Theme-aware CSS (rendered from `.tpl`) |

## Refresh

```bash
erch refresh waybar
```
